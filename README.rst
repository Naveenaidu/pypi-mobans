Installation
================================================================================

You can install it via pip:

.. code-block:: bash

    $ pip install setupmobans


or clone it and install it:

.. code-block:: bash

    $ git clone http://github.com/moremoban/setupmobans.git
    $ cd setupmobans
    $ python setup.py install
