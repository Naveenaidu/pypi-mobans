Installation
================================================================================

You can get it:

.. code-block:: bash

    $ git clone http://github.com/moremoban/setupmobans.git


Notes
================================================================================


In order to run, `python setup.py publish`, you will have setup `.pypirc` in
your home folder as::

   [distutils]
   index-servers =
     pypi

   [pypi]
   username=your_name
   password=your_password


And you need `gease` installed too.
